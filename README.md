# swarmdog

bzz自动兑换支票程序。[下载地址](https://github.com/muzexi3000/swarmdog/releases/download/v1.5/SwarmDog.zip)
* 自动兑票
* 导出Excel
* 运行一次程序后会在程序目录下生成data.json配置文件，可以修改SleepSeconds配置每隔多少秒钟遍历一次兑换
* 兑换报错，可能是bee节点需要梯子，或者可以在swarmdog上选择忽略异常
